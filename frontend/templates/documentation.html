System Documentation

Comprehensive documentation of the expert system's variables, rules, and architecture

Input Variables

Patient Profile

Variable Type Values
Age Group Selection Child / Adult / Elderly
Allergy Selection None / Peanut / Seafood

Symptoms

Variable Type Values
Itching Selection Yes / No
Burning Sensation Selection Yes / No
Pain Selection Yes / No

Lesion Appearance

Variable Type Values
Appearance Multiple Selection Circular, Annular, Scaly / Flaky, Well-Demarcated, Swollen, Papules / Pustules, Crusting / Oozing, Hyperpigmented / Depigmented, Ulcer
Lesion Size Selection Larger than 5mm / Smaller than 5mm

Output Variables

Variable Type Possible Values
Disease Inference Result Eczema, Contact Dermatitis, Ringworm, Acne Vulgaris, Skin Ulcer
Treatment Recommendation Topical Corticosteroid, Moisturizer Therapy, Antifungal Cream, Antibiotic Ointment, Pain Relief Medication
Lifestyle Recommendation Avoid Sharing Personal Items, Maintain Skin Hygiene, Avoid Scratching, Avoid Irritants, Regular Wound Care, Reduce Sun Exposure, Seek Medical Attention
Diet Recommendation Avoid Peanut Products, Avoid Seafood, Anti-Inflammatory Diet, Low Sugar Diet, High Protein Diet, Adequate Hydration

Rule Layers

Layer 1 Disease Identification

Disease Condition Contagious
Eczema IF Itching AND (Allergy OR NOT Allergy) AND (Scaly OR Hyperpigmented OR Small_Lesions) No
Contact Dermatitis IF Burning_Sensation AND Allergy AND (Well_Demarcated OR Crusting OR Swollen) No
Ringworm IF Itching AND (Circular OR Annular OR Scaly) Yes
Acne Vulgaris IF Papules_Pustules AND (Small_Lesions OR NOT Pain) No
Skin Ulcer IF Ulcer AND Pain AND Large_Lesions No

Layer 2 Suggested Treatment

Condition Treatment
IF Disease = Eczema AND Large_Lesions Topical Corticosteroid
IF Disease = Eczema AND Small_Lesions Moisturizer Therapy
IF Disease = Ringworm Antifungal Cream
IF Disease = Acne_Vulgaris AND Small_Lesions Antibiotic Ointment
IF Disease = Contact_Dermatitis Topical Corticosteroid
IF Disease = Skin_Ulcer AND Pain Antibiotic Ointment
IF Pain AND NOT Disease = Acne_Vulgaris Pain Relief Medication

Layer 3 Suggested Lifestyle

Condition Lifestyle Recommendation
IF Disease = Ringworm Avoid Sharing Personal Items OR Maintain Skin Hygiene
IF Disease = Eczema AND Age_Group = Child Avoid Scratching
IF Disease = Eczema AND Age_Group = Adult Avoid Irritants
IF Disease = Eczema AND Age_Group = Elderly Regular Wound Care
IF Disease = Acne_Vulgaris AND Age_Group = Adult Reduce Sun Exposure
IF Disease = Contact_Dermatitis Avoid Irritants
IF Lesion_Size = Large AND NOT Disease = Acne_Vulgaris Seek Medical Attention
IF Age_Group = Elderly AND Pain Regular Wound Care

Layer 4 Diet Recommendation

Condition Diet Recommendation
IF Allergy = Peanut Avoid Peanut Products
IF Allergy = Seafood Avoid Seafood
IF Disease = Eczema AND NOT Allergy Anti-Inflammatory Diet
IF Disease = Acne_Vulgaris Low Sugar Diet
IF Disease = Skin_Ulcer High Protein Diet
IF Age_Group = Elderly AND NOT Disease = Acne_Vulgaris High Protein Diet
IF Age_Group = Child AND NOT Allergy Adequate Hydration

Key Components

Component Description
Knowledge Base Contains all rules, input definitions, and output variables. Rules are organized into 4 layers for different types of inference.
Inference Engine Uses forward chaining algorithm to evaluate rules layer by layer. Applies pattern matching to determine applicable rules.
User Interface Web-based interface built with Flask. Provides forms for input collection and displays diagnosis results.
Explanation Facility Tracks which rules fired during inference and displays them to the user for transparency.
